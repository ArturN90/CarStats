# CarStats

A simple tool to make some statistic based on the data scraped from otomoto.pl. 

# Building on Ubuntu
## 1. Install required dependencies:  
  a) Download supported geckodriver for Firefox browser  
  ~~~
  cd depends  
  make
  ~~~  
## 2. Install setup.py  
~~~
python setup.py install
~~~
